OBSTACLE DETECTOR MACHINE (IoT Arduino Project)

SYNOPSIS
Abstract

Obstacle detector machine is the IoT related project that mainly can be used to detect the distance from one object to another and triger a buzzer alarm to notify that there is an obstacle in which a given object is going to crash so as to keep it on a safe side.

Name Of The Project: OBSTACLE DETECTOR MACHINE

Objectives:

  • To calculate the distance from one object to another
  • To notify the obstacles that an object is going to face
  • To triger the buzzer before an object crushes
Platform: Ubuntu OS

Technologies used:

  • C++
  • c#
Software Requirements:
  • Microsoft Windows or Linux (Ubuntu)
  • Arduino IDE
  • C++ or C# Programming Language
Hardware Requirements:
  • Intel Pentium IV Processor or Equivalent or Higher
  • 512 MB RAM or Higher
  • 20GB HDD or Higher
  • Network Connectivity
  • Arduino UNO or NANO or Whichever
  • Jumper Wires
  • Buzzer (Alarm)
  • Bread Board
  • LDE
  • Source Of Power (Not More Than 12DC Volt
